Projects

C, C++, Administration Linux System, Network, Devops And Web App Projects.

This Section Contain All The Projects That I Made During My Curriculum At 1337 Coding School

FT_TRANSCENDANCE

Web Project that aims to create of a Ping Pong SPA game with several functionalities as Chat, 2FA Authentication, Oauth2 authentication and etc using Angular and Tailwind CSS in Front-End and NestJs for Back-End.

FT_IRC

Creation of an IRC Server in C++ (Internet Relay Chat) based on the IRC protocol, which is a Protocol used by IRC servers and clients, IRC Protocol is a text-based TCP protocol.

INCEPTION

This project aims to expand our knowledge of system administration through the use of Docker. We have virtualized several Docker images using Docker-compose and creating them in a virtual machine.

FT_CONTAINERS

implementation of STL data structures in C++, such as vectors, lists and maps, to understand how they work and to learn more about C++ OOP.

CUB3D

graphical project in C aimed at repleimenting a famous game from the 90s (wolfstein3d) with a minilibx (minimal C library for graphical project) using raycasting Method.

NETPRACTICE

This project is a comprehensive guide to understanding the fundamentals of networking. It covers a wide range of topics, including network devices, the OSI model , subnet masks, routers, switches, and other related topics.

MINISHELL

Creation of a simplified command shell in C, capable of executing builtin commands, no builtin commands and managing pipes, redirects and heredocs.

FDF

The goal of this C graphics project is to create a 3D wireframe representation of a 2D map with the help of a minimal graphics library.

MINITALK

This is a C project that uses interprocess communication via Posix signals. In fact, it's a simple client-server communication system implemented using signals in C. The client sends a message to the server, and the server receives and displays the message character by character.
